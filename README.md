# 🚀 MultiShop Mikroservis E-Ticaret Projesi

## 📌 Proje Hakkında

MultiShop projesi, modern mikroservis mimarisi kullanılarak geliştirilmiş bir e-ticaret backend sistemidir. Toplamda 8 mikroservisten oluşmaktadır. Proje, dağıtık sistem mimarisi prensipleri uygulanarak tasarlanmıştır.

Bu proje, ASP.NET Core kullanılarak gerçek dünya e-ticaret senaryolarını simüle eden bir backend geliştirme deneyimi sunmaktadır.

## 🎯 Projenin Amacı

* Mikroservis mimarisini uygulamalı olarak öğrenmek
* Modern backend teknolojilerini kullanmak
* Dağıtık sistem mantığını kavramak
* Güvenli kimlik doğrulama sistemi oluşturmak

## 🏗️ Mimari Yapı

* Onion Architecture
* CQRS Pattern
* Mediator Pattern
* Repository Pattern
* SOLID Prensipleri

## ⚙️ Kullanılan Teknolojiler

* ASP.NET Core Web API
* Docker
* Redis
* MongoDB
* PostgreSQL
* MSSQL
* Dapper
* SignalR
* RabbitMQ
* Swagger & Postman
* Identity Server
* JWT Authentication & Authorization
* Ocelot API Gateway

## 🔐 Güvenlik

* Role-based authorization
* Token tabanlı kimlik doğrulama
* Identity Server entegrasyonu

## 🌐 Mikroservisler

Proje toplamda 8 mikroservisten oluşmaktadır:

* Catalog Service
* Basket Service
* Order Service
* Discount Service
* Payment Service
* Cargo Service
* Comment Service
* Identity Service
